# Exercise 41 - Employee Management System

## Objective

Use classes, dictionaries, lists, and JSON file handling together.

## Task

Create an employee management system that saves employee data to a JSON file and loads it back.

## Instructions

- Create an `Employee` class with `__init__` and `__str__`.
- Store employees in a dictionary.
- Save employee data to `emps.json`.
- Load the data back from the file.
- Print all employee details.

## Project Structure

- `Exercise_41.py` - Python source code
- `emps.json` - JSON file containing employee data
- `41_output.png` - Program output screenshot

## Expected Output

```
Employee Records:
ID: E101 | Name: Alice | Salary: 65000
ID: E102 | Name: Bob | Salary: 72000
ID: E103 | Name: Charlie | Salary: 58000
```
